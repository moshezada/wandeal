# איך להעלות את Wandeal ל-Google Play (Flutter) — מדריך מלא

יש שתי דרכים להגיע ל-Play. בחר את מה שמתאים לך.

---

## דרך א' — הכי קלה: PWABuilder (ממליץ להתחיל מכאן)
מאחר שאפליקציית ה-Web שלך **כבר חיה**, אפשר לעטוף אותה לקובץ .aab בלי לבנות Flutter בכלל:
1. היכנס ל-https://www.pwabuilder.com
2. הדבק את כתובת האפליקציה החיה (Wandeal) ולחץ Start.
3. בחר **Android** → **Generate Package** → הורד את קובץ ה-.aab.
4. העלה את ה-.aab ל-Google Play Console.

יתרון: בלי קוד, בלי התקנות, דקות ספורות.

---

## דרך ב' — Flutter דרך בנייה בענן (Codemagic, חינם)
לאפליקציה native מלאה מקוד ה-Flutter, בלי להתקין כלום במחשב:

1. **העלה את תיקיית `flutter-app` ל-GitHub** (צור repo חדש והעלה אליו את הקבצים).
2. היכנס ל-https://codemagic.io והירשם (חינם, 500 דקות בנייה בחודש).
3. חבר את ה-repo. Codemagic יזהה את `codemagic.yaml` שכבר נמצא בתיקייה.
4. **חתימה (חובה ל-Play):** ב-Codemagic → Android code signing → צור/העלה Keystore (Codemagic מסביר צעד-צעד). בלי חתימה, Play לא יקבל את ה-.aab.
5. הרץ את ה-workflow **wandeal-android** → בסיום הורד את קובץ ה-`app-release.aab`.

---

## העלאה ל-Google Play Console (משותף לשתי הדרכים)
1. `play.google.com/console` → **צור אפליקציה** (שם: Wandeal, אפליקציה, חינם).
   - הערה: בעבר יצירת אפליקציה נכשלה אצלך עם שגיאת שרת — אם זה חוזר, נסה שוב מאוחר יותר או פנה לתמיכת Play.
2. מלא את דף החנות: תיאור, צילומי מסך, אייקון, קטגוריה, מדיניות פרטיות (חובה).
3. **Production → Create release → העלה את ה-.aab**.
4. מלא דירוג תוכן (Content rating) ו-Data safety.
5. שלח לבדיקה. אישור ראשון לוקח בדרך כלל מספר ימים.

## דברים שצריך מראש
- חשבון Google Play ($25 חד-פעמי — כבר שולם בחשבון שלך).
- מדיניות פרטיות נגישה (אפשר דף פשוט באתר).
- גילוי נאות שהאפליקציה מכילה קישורים שיווקיים (אפיליאט).

## הערה כנה
הקמפול, החתימה וההעלאה מתבצעים בסביבת בנייה ובדפדפן שלך — לא ניתן להפיק קובץ Play חתום או להעלות אותו מתוך הצ'אט. המדריך והקובץ `codemagic.yaml` הופכים את זה לכמה לחיצות.
